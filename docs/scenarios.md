# Scénarios fonctionnels

## Scénario 1: Export d'un CSV vide

- Étant donné une liste de transactions vide.
- Quand l'utilisateur clique sur le bouton Exporter en CSV.
- Alors le fichier téléchargé contient uniquement la ligne d'en-tête.

## Scénario 2: Addition dans le module calculator

- Étant donné la fonction add du module calculator.
- Quand on lui passe les valeurs 2 et 4.
- Alors le résultat retourné est 6.
